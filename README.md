# 라이브러리 썸네일 메이커

FLO 내 리스트용 썸네일 이미지를 쉽게 만들 수 있는 웹 애플리케이션입니다.

## 기능

- 📐 2가지 레이아웃 선택 (1장 또는 2~4장)
- 🖼️ 이미지 선택 및 미리보기
- 🎨 배경색 선택 (프리셋 + 커스텀)
- 👆 드래그로 이미지 위치 조정
- 💾 고품질 JPG 다운로드
- 🎵 FLO 내 리스트로 바로 연동

## 사용 방법

1. 레이아웃 선택
2. 사진 선택
3. 배경색 및 이미지 위치 조정
4. 썸네일 생성 후 FLO에 업로드

## 기술 스택

- HTML5 Canvas
- Vanilla JavaScript
- CSS3 (Pretendard 폰트)
- 반응형 디자인 (모바일 우선)

## 배포

이 프로젝트는 정적 웹사이트로, Netlify, Vercel, GitHub Pages 등에서 배포 가능합니다.
